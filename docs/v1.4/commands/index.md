# Commands

This section documents CodeNOW CLI commands for the Data Plane workflow. 

## Command Index

### Data Plane (dp)

- **bootstrap**: [dp bootstrap](dp/bootstrap.md) — Bootstraps the Kubernetes cluster for Data Plane installation.
- **dashboard**: [dp dashboard](dp/dashboard.md) — Displays a management dashboard with the current state of the Data Plane.
- **config encrypt**: [dp config encrypt](dp/config.md) — Encrypts secret values in a plaintext operator configuration file.
